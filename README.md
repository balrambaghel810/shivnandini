## Shivnandani Creations — E-commerce (Frontend)

Premium, responsive multi-page e-commerce website for **Shivnandani Creations By Manisha Sharma** (Hand Woven Designers).

### Pages
- `index.html` (Home)
- `shop.html` (All Products)
- `sarees.html` (Saree Collection)
- `product.html` (Product Details)
- `cart.html` (Cart)
- `about.html` (About Us)
- `contact.html` (Contact)

### Tech
- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- Motion (Framer's web animation library) for smooth animations

### How to run
You can open `index.html` directly in a browser.

For best results (recommended), run a local server:

```bash
npx serve
```

Then open the shown local URL (and navigate between pages).
